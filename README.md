# PrAha 管理コンソール（モック）

課題・チーム・進捗を管理する社内向け管理画面の **UIモック** です。
「こんな感じでどうですか？」を確認するためのもので、**DB・API・永続化は未実装**（変更はブラウザ上のstateにのみ反映されます）。

## 画面

左サイドバーから3つのページに移動できます。

| メニュー | 内容 |
| --- | --- |
| **ユーザー** (`/users`) | 受講生の一覧。所属している期・メールアドレス・チームを変更できる。期を変えるとチームの選択肢がその期のものに切り替わる。 |
| **課題開放** (`/challenge-access`) | チーム × 課題ジャンルのマトリクス。どのチームがどのジャンルの課題を閲覧できるかをトグルで設定する（ジャンル単位。列見出しから一括切り替えも可）。 |
| **進捗一覧** (`/progress`) | 期セレクターで期を選ぶと、その期の受講生が対象になる。縦軸に課題（ジャンルごとにグルーピング）、横軸に受講生を並べ、どの課題まで進んでいるかをステータス（完了 / レビュー中 / 着手中 / 未着手）で表示する。 |

## 起動

```bash
npm install
npm run dev
# http://localhost:3000 （/users へリダイレクト）
```

## 構成

- Next.js 16 (App Router) / TypeScript / Tailwind CSS v4
- モックデータは [`lib/mock-data.ts`](lib/mock-data.ts) に集約（期・チーム・ユーザー・課題ジャンル・課題・課題開放・進捗）
- 各ページはクライアントコンポーネントで、操作は `useState` 上でのみ反映
